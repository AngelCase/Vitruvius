## 累積和とは
配列$a_0,a_1,a_2,...,a_{N-1}$に対して
配列$s_0,s_1,s_2,...,s_{N-1},s_N$を以下のように定めたもの。
- $s_0 = 0$
- $s_1 = a_0$
- $s_2 = a_0 + a_1$
- $s_3 = a_0 + a_1 + a_2$
- ...
- $s_N = a_0 + a_1 + a_2 + ... + a_{N-1}$

例えば、$a = {3,6,8,2}$であれば$s={0,3,9,17,19}$である。

## 使い道
これを用いると、例えば配列$a$の区間$[3,7)$の総和を以下で求められる：
